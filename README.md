# AL MYTHAQ Building Cleaning Service (Frontend)

Tagline: **Cleaning Beyond Expectations**

This is a frontend-only Next.js 14 (App Router) project for **AL MYTHAQ Building Cleaning Service**.

## Tech

- Next.js 14+ (App Router)
- TypeScript
- Tailwind CSS
- shadcn/ui-style components (Radix UI + utility classes)

## Getting Started

1. Install dependencies:

```bash
npm install
```

2. Run the dev server:

```bash
npm run dev
```

Open http://localhost:3000

## Project Notes

- Frontend-only: no database, auth, or payments.
- Contact and booking forms are mocked and show success toasts.
- Booking page includes WhatsApp integration with a prefilled message.
